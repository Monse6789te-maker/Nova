# ⚡ Nova - Detector de Residuos con IA


## 🌍 Descripción

Nova es una aplicación web progresiva que utiliza inteligencia artificial (MobileNet) para detectar y clasificar residuos en tiempo real, ayudándote a reciclar correctamente mientras calculas tu ahorro energético.

## ✨ Características

- 🤖 **IA en tiempo real** - MobileNet con +90% de precisión
- ♻️ **Clasificación inteligente** - Orgánico, Reciclable, No Reciclable
- ⚡ **Cálculo de ahorro energético** - Visualiza tu impacto en kWh
- 🎮 **Gamificación completa** - Niveles, logros, rachas
- 📊 **Comparación social** - Compárate con otros usuarios
- 🌙 **Modo oscuro** - Diseño adaptable
- 📱 **100% Responsive** - Funciona en móviles y desktop
- 🔒 **Privacidad total** - Todo se procesa en tu dispositivo
